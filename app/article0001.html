Gulp 简介

Gulp.js 是一款流式前端构建工具，其使用流程也是十分的简单易懂，下面我们简单介绍一下如何使用Gulp。

1. 全局安装Gulp：

如果你之前已经安装过Gulp的话，请运行 npm rm --global gulp 来确保老版本的Gulp已经被删除。

$ npm install --global gulp-cli

2. 初始化你项目

$ npm init

3. 作为项目的devDependencies安装Gulp

$ npm install --save-dev gulp

4. 在你项目的根目录创建一个 gulpfile.js:

      var gulp = require('gulp');
        gulp.task('default', function() {
        // code here
      });

5. 运行Gulp:

$ gulp
